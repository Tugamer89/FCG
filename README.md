This repository contains the laboratory exercises for the **Computer Graphics Fundamentals** (Fondamenti di Computer Grafica - FCG) course. These labs are designed to explore and implement theoretical concepts learned during lectures.

> [!NOTE]  
> This repository is public primarily to take advantage of free GitHub Actions (CI/CD) and for easy sharing with friends.

---

## Project Structure

- **Lab 1: Text Editor** - Basic text manipulation and echo.
- **Lab 2: Drawing Shapes** - Rendering primitive geometric shapes.
- **Lab 3: Breakout** - A "Block Breaker" game implementation exploring game loops and physics.

## Requirements & Build

The project uses **CMake** as the build system and targets **C++**.

```bash
# Clone the repository
git clone [https://github.com/tugamer89/fcg.git](https://github.com/tugamer89/fcg.git)
cd fcg

# Create a build directory
mkdir build && cd build

# Configure and build
cmake ..
cmake --build .
```
